Operadores:
 -Aritiméticos
        5+2 é 7
        5-2 é 3
        5*2 é 10
        5/2 é 2.5
        5%2 é 1
        5**2 é 25 // 5² é 25  
     
 -Atribuição 
        Precedência
        () ** * / % + - 
        ---------------->
        // ordem de importância da esquerda p/ direita
        // ** é elevação, expoente.

 -Relacionais
    ‹ › ‹= ›= == != 
    // != é não igual ou diferente. 
    // === identicos e mesmo tipo.
    // == é igual. 

        5 == 5    true 
        5 == '5'  true 
        5 === '5' false // diferente, são tipos diferentes. valor interno é o mesmo.
        5 === 5   true // pois são identicos
 
 -Lógicos
    !   // quer dizer negação    (não) 
    &&  // quer dizer conjunção  (e) 
    ||  // quer dizer disjunção  (ou) 

        Negação
        ! true -> false // não verdadeiro é falso
        false -> true // não falso é verdadeiro 
    
        Conjunção // valor binario que tem 2 logicos. Basta que um deles seja falso tudo é false 
        true  && true  -> true   // Ex.: eu tenho 1 caneta red e 1 azul eu fico satisfeito, é true 
        true  && false -> false  // Ex.: Qualquer outra combinação de canetas red, azul, verde, é false 
        false && true  -> false  // Ex.: Combinação 2 red ou 2 azuis   é false
        false && false -> false  // Ex.: 2 falsas é false 
  
        Disjunção // Basta um valor seja verdadeiro tudo vira verdadeiro.
        true  || true  -> true  // Ex.: 1 valor seja verdadeiro tudo vira verdadeiro.
        true  || false -> true  // Ex.: 1 valor seja verdadeiro tudo vira verdadeiro.
        false || true  -> true  // Ex.: 1 valor seja verdadeiro tudo vira verdadeiro.
        false || false -> false // Ex.: 2 falsos tudo false  
    
        Exp.: 
            var a = 5
            var b = 8
            a > b && b % 2 == 0 
            // quando tenho Aritiméticos, relacionais e Logicos na mesma expressão 
            // Primeiro faz os Aritiméticos %
            // Segundo os Relacionais > ==
            // Terceiro os Logicos &&
            Sendo b(8) % 2  dá 4
            4 == 0 dá  4 resto 0
            é verdadeiro
            Sendo a(5) > b(8)  5 não é maior que 8 
            é false falso 
            falso && verdadeiro  é false falso 
    
            a ‹= b || b / 2 == 2
            Sendo b(8) / 2  dá 4
            4 == 2  4 não é igual a 2 
            é false
            Sendo a(5) ‹= b(8)  5 é menor ou igual a 8 
            verdadeiro || false  é true verdadeiro   
        Se na mesma expressão tiver um && (e), || (ou), ! (não), segue a sequencia certa: 
        1º ! (nao), depois o && (e), depois || (ou)      
  
            idade ‹= 15 && idade ›= 17 // a idade está entre 15 e 17? 
            estado == 'RJ' || estado == 'SP'            // o estado é RJ ou SP?
            salário > 1500 && sexo != 'M'            // o salário é maior de 1500 e não é Masculino?
    
        Precedência
    Primeiro faz os Aritiméticos   () ** * / % + - 
    Segundo os Relacionais         ‹ › ‹= ›= == != 
    Terceiro os Logicos            ! (nao), && (e), || (ou)
